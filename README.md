## Customizing IdeaVim Keys

1. Press <kbd>CMD</kbd>+<kbd>SHIFT</kbd>+<kbd>A</kbd>

2. Select

```sh
IdeaVim: Track Action IDs
```

We can add custom bindings to our `~/.ideavimrc`

```sh
nmap S <action>(AceWordStartAction)
```
